# IoT-Mini Project 10

ESP8266 dihubungkan dengan ESPServer untuk mengambil ID CARD yang dideteksi oleh RFID kemudian akan menampilkan ID CARD dari tersebut ke dalam website

--------------------------------------------------

### Hardware yang digunakan: 
- NodeMCU ESP8266
- RFID
- Kabel Jumper
- Project Board

### Software yang digunakan: 
- Arduino IDE
